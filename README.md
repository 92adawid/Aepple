# Återskapa Apple.com – Inlämningsuppgift 1

**Projektmedlemmar:**  
- Adam (startsida / landningssida)  
- Arnell (supportsida)

**Vald webbplats:** [Apple.com](https://www.apple.com)  
**Originalets URL:** https://www.apple.com

## Beskrivning
Vi har återskapat utseendet och strukturen från Apples webbplats med fokus på det visuella och responsivitet. Interaktivitet har inte varit prioriterad – istället har vi lagt krutet på att få färger, typografi, mellanrum och layout att kännas så nära originalet som möjligt, oavsett skärmstorlek.

## Arbetsfördelning
- **Adam** ansvarade för den stora landningssidan (startsidan) där bland annat global navigering, hero-sektioner, produktkort och footer återskapades.  
- **Arnell** byggde en supportsida med en layout som följer Apples designspråk – med liknande header, tydliga kategorier och responsiv anpassning.

## Avgränsningar
Eftersom apple.com har väldigt mycket innehåll gjorde vi följande avgränsningar:
- Vi återskapade **endast startsidans visuella utformning** (Adam) samt en **översiktlig supportsida** (Arnell), inte varje undersida.
- Interaktiva element (sökfunktionens fullständiga beteende, varukorgshantering etc.) har förenklats.
- Vi använde CSS återgivning av ikoner via Font Awesome istället för Apples egna symboler.

## Tekniker
- HTML5 & CSS3  
- Font Awesome (för ikoner)  
- Flexbox & CSS Grid för layout  
- Media queries för responsivitet (mobil, tablet, desktop)  
- Stöd för mörkt läge (prefers-color-scheme)  
- Git & GitHub för versionshantering
